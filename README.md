# Mokio
_______

Mokio is a library that extends Okio with a variety of useful features. It includes:
- Process Management: Launch and control processes with ease.
- Environment Variables: Get and set system environment variables.
- File Utilities: Includes a FileMode structure and a Stat implementation.
- Enhanced Symlink Support: Create and manage symbolic links as-is, with support for Android 5.0+ and MingwX64.
